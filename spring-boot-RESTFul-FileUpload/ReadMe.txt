Step 1: Checkout the Project folder - https://github.com/shyamuserid/spring-boot-RESTFul-FileUpload.git
Step 2: Navigate to the local Project Folder and run below command to bootstrap the app.
        mvn spring-boot:run
Step 3: Open a browser and enter URL localhost:8080/
Step 4: Click the 'Choose File' Button and select a file that needs to be uploaded.
Step 5: Then Click "Upload" Button.
Step 6: Check the file on the Desktop folder
